## Setup

This project uses [pnpm](https://pnpm.io/installation). Please install it if you haven't.

Then install dependencies to finish the setup.

```
pnpm install
```

## Development

- `pnpm test` to run tests.
- `pnpm lint` to run linting and type-checking.
- `pnpm build` to build.
